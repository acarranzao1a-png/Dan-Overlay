# Dan-Overlay

Este repositorio se utiliza **únicamente** para publicar y distribuir el ejecutable `.exe` de Dan-Overlay a través de **GitHub Releases**.

## Descarga

Puedes descargar la última versión del ejecutable desde la sección de [Releases](../../releases/latest).

## Notas

- Este repositorio no contiene código fuente.
- Solo se usa para gestionar las versiones publicadas del ejecutable.
